Подробное описание

Триангуляция массива треугольников

Аргументы
srcVertex[in/out] Указатель на входную структуру треугольников. Перезаписывается разбитыми треугольниками
srcCount[in/out] Количество входных треугольников. Четное число. Должно быть меньше реального размера srcVertex хотябы в 2 раза
maxWidth[in] Ограничение по ширине
maxHeight[in] Ограничение по высоте
dstSuitableVertex[out] Указатель на выходную структуру годных треугольников
srcTreatedCount[out] Указатель на число годных треугольников

void
